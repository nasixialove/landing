## 使用说明

### 如何查看网站
1. 直接双击打开 `index.html` 文件
2. 或者使用本地服务器(推荐):
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve
   ```

### 如何修改内容
- 文字内容: 编辑 `index.html`
- 样式调整: 编辑 `styles.css`  
- 图片替换: 直接替换对应的图片文件

建议使用 TinyPNG (https://tinypng.com/) 压缩图片:
- 可以减少50-80%的文件大小
- 不会明显降低图片质量
- 提升网页加载速度
