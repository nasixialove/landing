# 快速测试指南

## 🚀 立即查看网站

### 方法1: 直接打开 (最简单)
1. 找到 `index.html` 文件
2. 双击打开
3. 网站会在浏览器中显示

### 方法2: 使用本地服务器 (推荐)
```bash
# 在项目文件夹中打开命令行,然后运行:
python -m http.server 8000
```
然后在浏览器打开: http://localhost:8000

## 🔧 常见问题

### Q: 图片没有显示?

- 娜絲夏logo.png
- 滑水晶缽.jpg
- 活動照.JPG

### Q: 样式看起来不对?
A: 确保 `styles.css` 文件在同一文件夹

### Q: Google表单打不开?
A: 检查链接是否为表单填写链接(不是编辑链接)
当前链接是编辑链接,需要改为公开填写链接

A: 网站已经适配响应式设计,在手机浏览器中应该正常显示
